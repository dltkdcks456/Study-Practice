🟤**Bronze 1단계 1546번 1회시도**

```python
import sys
n = int(sys.stdin.readline())
n_list = list(map(int,sys.stdin.readline().split()))
n_new_list = [score / max(n_list) * 100 for score in n_list]
print(sum(n_new_list) / n)
```
