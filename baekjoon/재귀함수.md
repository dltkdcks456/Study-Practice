# 재귀함수

🟤**Bronze 5단계 10870번 1회시도**

```python
def pibo(n):
    if n == 0:
        return 0
    elif n == 1:
        return 1
    else:
        return pibo(n-1) + pibo(n-2)

print(pibo(int(input())))
```



🟤**Bronze 5단계 10870번 1회시도**

```python
def fac(n):
    if n==0 or n == 1:
        return 1
    else:
        return n * fac(n - 1)

print(fac(int(input())))
```
